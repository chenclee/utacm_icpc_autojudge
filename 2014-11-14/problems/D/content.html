Palindromes

You're given a string of characters. How many ways can you arrange these characters to make a palindrome?

A palindrome is a string of characters that can be read forwards and backwards the same. For example, 'racecar' and 'noon' are palindromes.

Input

The first line of input will contain a single integer T, the number of test cases. The first line of each test case has an integer, N, the size of the string. The next line contains a single string (no whitespace) of size N.

Constraints

There are at most 100 test cases. The size any string will not exceed 20 for each.

Output

For each test case, print the number of ways that string could be arranged into a palindrome.

Sample Input

4
2
oo
3
OoO
4
abab
4
abac

Sample Output

1
1
2
0
