Math Test

You're absolutely terrible at math. You don't understand what the need for math is if you have a computer. So... As a result, you have failed your math test. However, your teacher knows that you're a star programmer, so she has given you an interesting compromise...

If you create a program to automatically grade the math tests, she will give you an A in the class. It's quite a fair trade, so you'll do your best to accomplish this.

You're very fortunate that tests are only over addition and subtraction, and maybe sometimes multiplication. The questions are always in the format of A OP B = C, where A and B are integers and OP is either '+', '-', or '*'. Determine the percentage of answers provided, C, are correct.

Input

The first line of input will contain a single integer T, the number of test cases you have
The first line of each test case contains an integer N, the number of questions on the test.
The following N lines have each question of the test which is in the format shown in the description.

Constraints

1 ≤ T ≤ 10
1 ≤ N ≤ 1000
1 ≤ A ≤ 100
1 ≤ B ≤ 100

Output

For each test case, print the percentage of correct answers on the test rounded to the nearest integer.

Sample Input

2
3
1 + 2 = 3
2 - 1 = 1
4 * 2 = 12
4
1 + 1 = 2
2 + 2 = 4
2 * 2 = 4
3 - 4 = -1

Sample Output

67%
100%
