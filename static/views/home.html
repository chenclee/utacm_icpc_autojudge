UT ACM Programming Competition

Welcome to the UT ACM's Programming Competition! Over the next few hours you will be solving a series of competitive programming questions using this website.

On the problems page you will find a list of the problems which you must solve. These problems are listed in no particular order and are all worth the same amount of points! There is not penalty for answering easier questions, and no bonus for answering harder questions. Your only goal should be to answer as many of these questions as possible.

Submitting Solutions

You will only receive a limited number of tries to correctly solve each problem. Once you click the "Attempt to Solve" button for a specific problem, a timer will begin counting down. While that timer is non-zero, you will be able to download the input text file and submit a corresponding output text file. You can do whatever you want within in the time limit, which means you are welcome to last-minute debug your program if you so choose, but you must upload the output and source code and click submit before the time runs out. When the timer runs out, you will have to use another attempt to solve the problem, but please note that the input file varies with each attempt! There is no penalty for submitting incorrectly, other than the loss of an attempt.

Requesting Clarifications

If you have any questions about a problem, please submit a clarification in the text box located below each problem statement. A judge will try to respond as quickly as possible, but be warned that the response will more often than not be for you to re-read the problem statement.

Scoring

Results can be viewed on the Scoreboard tab. Each row on the scoreboard represents a contestant that has solved or attempted at least one problem, and are ordered by descending rank. Ranks are determined by the total number of problems solved, with ties being broken by the total amount of time taken to produce the correct solutions.
